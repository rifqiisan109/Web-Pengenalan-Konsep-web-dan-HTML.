Pengenalan HTML

HTML adalah bahasa markup untuk membuat struktur halaman website.
HTML Bukan Bahasa Pemrograman

HTML hanya digunakan untuk menyusun dan menampilkan konten pada halaman web.
File HTML

File HTML biasanya memiliki ekstensi .html atau .htm.
Struktur Dasar HTML

HTML memiliki struktur dasar seperti DOCTYPE, html, head, dan body.
Tag HTML

Tag html, head, dan body adalah bagian utama dalam dokumen HTML.
Tag dalam HTML

Tag digunakan untuk menandai fungsi suatu bagian pada halaman web.
Elemen dan Atribut

Elemen terdiri dari tag pembuka, isi, dan tag penutup.
Format Teks

HTML menyediakan berbagai tag untuk mengatur tampilan teks.
Gaya Teks

HTML memiliki berbagai tag untuk mengubah gaya tulisan.
Karakter Khusus

HTML menyediakan berbagai simbol khusus dan fitur komentar.
Kesimpulan

Dengan memahami HTML, seseorang dapat mulai membuat halaman web sederhana.
